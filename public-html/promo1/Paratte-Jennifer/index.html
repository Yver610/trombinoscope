Petite présentation

Après sept ans d'expérience dans la peinture en bâtiment, j'ai ressenti le besoin d'une reconversion dans la région, l'envie d'apprendre et de me former dans un nouveau domaine. C'est pourquoi, je me suis tourné vers le métier de développeur web qui, en plus d'être très intéressant et passionnant, est un métier d'avenir. Étant une personne très sociable et souriante, le milieu de ce métier me correspond.Je m'adapte rapidement, aime le travail d'équipe, communiquer avec les autres et rester positive.

Compétences acquises

Créer des pages web en HTML, CSS, PHP7 et MySQL

Le minichat est composé d'une interface en HTML, CSS et Javascript, permettant de le personnaliser. Les messages sont dynamiquement envoyés et affichés en PHP grâce la base de données MySQL.

Utiliser les requêtes AJAX pour éviter le rechargement de page

Il utilise des requêtes AJAX en Javascript pour passer de simple page web à véritable application. L'execution périodique de la requête AJAX permet un rafraichissement régulier des messages, toutes les deux secondes.

Programmation Orientée Objet en PHP
