Compétences acquises

Créer des pages web en HTML, CSS, PHP7 et MySQL

Le mini chat est composé d'une interface en HTML, CSS et Javascript.
Les messages sont dynamiquement envoyés et affichés en PHP grâce la base de données MySQL.

Utiliser les requêtes AJAX pour éviter le rechargement de page

Le minichat utilise des requêtes AJAX en Javascript pour passer de simple page web à véritable application. L'execution périodique de la requête AJAX permet un rafraichissement régulier des messages, toutes les deux secondes.
